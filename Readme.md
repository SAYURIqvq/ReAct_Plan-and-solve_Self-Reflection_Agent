# ReAct_Plan-and-solve_Self-Reflection_Agent

一个轻量级、模块化的 LLM Agent 系统，融合推理（ReAct）、任务规划（Plan-and-Solve）与自反思机制（Self-Reflection），支持多步骤任务的自主执行与持续优化。

---

## 近期更新

### 模型与系统更新

- **模型服务切换**  
  主力 LLM 已从 DeepSeek 切换至 **Qwen（通义千问）**，支持更长上下文。

- **认证方式更新**  
  使用阿里云百炼平台 `DASHSCOPE_API_KEY` 作为统一认证方式。

- **文档与工具同步**  
  `README.md`、`.env` 示例、环境配置及工具调用方式已全面更新并对齐。

- **兼容性保障**  
  原有功能与工具链保持兼容，无需修改代码即可升级使用。

> 提示：首次运行时，系统会自动检测 `.env` 并引导完成 Qwen 配置。

## Web UI 功能

### 核心能力

- **Web 界面支持**  
  基于 `Chainlit` 构建现代化交互界面（`app.py`）

- **流式响应与过程可视化**  
  实时展示模型推理与执行过程

- **快捷指令支持**  
  提供预设 Starter，一键触发常用任务

### 配置与交互

- **自动环境检测**  
  首次访问自动检查配置状态

- **交互式配置向导**  
  支持 Web 端配置 API Key

- **文件管理能力**  
  支持拖拽上传文件至 `workspace/`，并自动纳入 Agent 工作流程

### 用户体验优化

- 支持任务中断（Cancel）与恢复  
- 工具执行过程可视化  
- 错误信息与系统提示清晰展示

## 启动方式

使用以下命令启动 Web 界面：

```bash
# 默认端口（仅本地访问）
chainlit run app.py

# 指定端口
chainlit run app.py -p 8080

# 指定访问地址
chainlit run app.py --host 192.168.1.100 --port 8000
```

访问：

```text
http://<host>:<port>
```

例如：

```text
http://127.0.0.1:8000
```

---

## IP 地址说明

- **127.0.0.1（localhost）**  
  仅本机访问，适用于本地开发环境（推荐）

- **局域网 IP（如 192.168.x.x）**  
  同一局域网设备可访问

- **公网 IP**  
  可对外访问，需要额外网络与安全配置
